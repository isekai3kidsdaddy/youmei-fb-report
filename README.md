# 悠美診所｜競業 FB 廣告分析週報

互動式儀表板，追蹤悠美診所與台灣主要醫美競業的 Facebook 廣告表現。

## 功能
- 📊 本週總覽：KPI 快照、發文數、互動率
- 🔥 競業亮點：本週最強貼文 Top 3
- 📈 互動率 & 六維度評分
- ⚔️ 療程競爭版圖
- ✅ 優點 / ❌ 缺點分析
- 💡 P1/P2/P3 改善建議
- 📌 下週行動清單
- 🔗 Meta 廣告庫直連

## 部署到 GitHub Pages

1. 建立新 repository（e.g. `youmei-fb-report`）
2. 上傳 `index.html`、`README.md`、`.nojekyll` 三個檔案
3. 進入 **Settings → Pages → Source → Deploy from a branch → main**
4. 儲存後約 1–2 分鐘，網址為：`https://<你的帳號>.github.io/youmei-fb-report/`

## 分析期間
2026/07/03 – 07/09

## 注意事項
- 本報告由 AI 根據即時抓取之 Facebook 公開資料自動產生
- 標 `*` 之競業本週未能完整抓取，情報定位沿用既有資料
- 請遵守台灣《醫療法》廣告相關規定
